source @ app-web/web-index.html, compiled to dist/web/index.html

Electron Renderer sources are in src/app-electron. Electron Main Process files are in run

  • Edit any file in src/app-web to force a full page reload
  • npm run dev compile/watch handled by webpack-dev-server
  • npm run electron compile/watch handled by main-appserver through webpack-dev-middleware
  • files are written to disk using WriteFilePlugin with filtering options to keep hot-reload modules from being written. These files can then be used as input for electron.
